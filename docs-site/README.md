# Auto Scan Backend Docs

Portal de documentacao do backend do Auto Scan construido com Docusaurus.

## Scripts

```bash
npm install
npm run start
npm run build
npm run serve
```

## Estrutura

- `docs/`: documentacao funcional e tecnica
- `src/pages/`: landing page do portal
- `docusaurus.config.ts`: branding, navbar, footer e plugins
- `sidebars.ts`: navegacao principal da documentacao

## Observacao

Os documentos foram reorganizados para refletir a arquitetura atual do backend e servir como base viva para onboarding, operacao e evolucao do produto.
